- WoT Extension Ontology

1. Introduction

This ontology provides additional Eclipse Ditto™ specific context for W3C Web of Things TDs and TMs.

2. Formats

Using this URL and "Content Negotiation" via Accept header, it is also possible to access the following different formats:

  • text/turtle
  • application/ld+json
  • application/rdf+xml

To access the formats the following links are also possible:

  • TTL: ditto-extension.ttl
  • JSON-LD: ditto-extension.jsonld
  • RDF: ditto-extension.rdf

3. Axiomatization

3.1 Datatype Properties

3.1.1 category

IRI: https://ditto.eclipseprojects.io/wot/ditto-extension#category

Optional category for WoT Property affordances, e.g. something like 'status' or 'configuration'.

Domain td:PropertyAffordance
Range schema:Text

3.1.2 deprecated

IRI: https://ditto.eclipseprojects.io/wot/ditto-extension#deprecated

Boolean flag indicating whether the Thing or affordance is deprecated.

Domain ditto:DeprecationNotice
Range schema:Boolean

3.1.3 supersededBy

IRI: https://ditto.eclipseprojects.io/wot/ditto-extension#supersededBy

A JSON Pointer (RFC 6901) referencing the replacement affordance that supersedes the deprecated one, e.g. '#/properties/newProperty'.

Domain ditto:DeprecationNotice
Range schema:Text

3.1.4 removalVersion

IRI: https://ditto.eclipseprojects.io/wot/ditto-extension#removalVersion

The semantic version (SemVer) in which the deprecated affordance will be removed, e.g. '2.0.0'.

Domain ditto:DeprecationNotice
Range schema:Text

3.2 Classes

3.2.1 DeprecationNotice

IRI: https://ditto.eclipseprojects.io/wot/ditto-extension#DeprecationNotice

A structured object indicating that a WoT Thing (ThingModel/ThingDescription), or an affordance (property, action, or event) is deprecated. Contains the following properties:

  • deprecated (boolean, required): Whether the Thing or affordance is deprecated
  • supersededBy (string, optional): JSON Pointer to replacement affordance
  • removalVersion (string, required): SemVer version when affordance will be removed

3.3 Object Properties

3.3.1 deprecationNotice

IRI: https://ditto.eclipseprojects.io/wot/ditto-extension#deprecationNotice

Indicates that a WoT Thing, or an affordance (property, action, or event) is deprecated. The value is a DeprecationNotice object containing deprecation details.

Domain td:Thing, td:PropertyAffordance, td:ActionAffordance, td:EventAffordance
Range ditto:DeprecationNotice
