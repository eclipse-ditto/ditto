# Code Reviews

This directory contains code review documents generated by Claude Code.

## Purpose

Store comprehensive code reviews for:
- Feature branches
- Pull requests
- Major commits
- Architecture changes

## Naming Convention

Format: `<pr-number>-code-review-<feature-name>.{md,pdf}`

Examples:
- `2287-code-review-partial-access-events.md`
- `2301-code-review-helm-pdb-fix.md`
- `2345-code-review-connection-pooling.md`

**Note**: Always start with the PR number for easy reference and sorting.

## Contents

Each review typically includes:
- Executive summary
- Critical/major/minor issues
- Architecture compliance analysis
- Recommendations
- Testing suggestions
- Rollout strategy

## Formats

Reviews are available in:
- **Markdown** (.md) - Source format, easy to read/edit
- **PDF** (.pdf) - Formatted for sharing and archival
